# Lognormal survivor function S(t)

Lognormal survivor function S(t)

## Usage

``` r
.S_lnorm(t, mu, sigma)
```

## Arguments

- t:

  Numeric vector of times.

- mu:

  Numeric scalar; lognormal location parameter.

- sigma:

  Numeric scalar; lognormal scale parameter.

## Value

Numeric vector the same length as `t`.
